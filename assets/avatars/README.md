# Plant Avatar Assets

This directory contains PNG images for the 8 plant personalities in GreenBuddy.

## Required Files

Create these PNG files (recommended: 400x400px, transparent background):

1. **cactus.png** - Dry, gruff personality
2. **orchidee.png** - Sophisticated, poetic personality
3. **monstera.png** - Cool, energetic personality
4. **succulente.png** - Neutral/default personality
5. **fougere.png** - Mysterious personality
6. **carnivore.png** - Playful personality
7. **pilea.png** - Friendly personality
8. **palmier.png** - Tropical personality

## Specifications

- **Format**: PNG with transparency
- **Size**: 400x400px (will be scaled by React Native as needed)
- **Background**: Transparent (RGBA)
- **Style**: Cute, plant-based, vector or illustrated
- **Quality**: 72 DPI minimum

## Implementation Notes

The PlantAvatar component requires these files to display the plant avatars. Without these images:
- The component will fail to load the image source
- Development will proceed with error handling
- Production builds will need actual images

You can use:
- AI image generators (DALL-E, Midjourney) for unique designs
- Open-source plant illustrations (OpenMoji, Font Awesome)
- Hand-drawn designs
- Vector graphics (Illustrator, Figma exports)
